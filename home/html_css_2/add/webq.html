WebP

Преимущества:

  1. Позволяет существенно снизить вес фотографии
  2. Поддерживает прозрачность
  3. Нет проблем с гранями и высококонтрастными местами

Недостатки:

  1. Не все браузеры поддерживают этот формат. Поддерживают: Google Chrome, Mozilla Firefox, Microsoft Edge, Opera, Android Browser, Chrome для Android, Samsung Internet Browser. (Не уверена в актуальности информации на сегодняшний день)

Однозначный победа JPG над WEBP. Ухудшается качество свечения и темного фона изображения. Перчинка остаётся без видимых изменений.

При почти равном размере изображения, JPG выигрывает у WEBP. Лучше детальность прорисовки пор кожи и правый зрачок.

WEBP оказался даже больше по размеру, чем JPG. Иллюстрация с огромным количеством мелких деталей.

При почти равном размере изображения, JPG одерживает победу над WEBP. Внимание на количество деталей мягкого белого фона.
